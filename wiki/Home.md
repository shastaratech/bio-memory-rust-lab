## Audience Paths

- [School track](https://github.com/shastaratech/bio-memory-rust-lab/blob/main/course/school-track.md):
  physical activities, short code, atom cards, graph games, and quantum wave cards.
- [University track](https://github.com/shastaratech/bio-memory-rust-lab/blob/main/course/university-track.md):
  Rust API design, graph algorithms, constrained search, and quantum simulation context.
- [Activity pack](https://github.com/shastaratech/bio-memory-rust-lab/blob/main/activities/classroom-games.md):
  classroom games that make types, memory, and graphs tangible.
- [Worksheet pack](https://github.com/shastaratech/bio-memory-rust-lab/blob/main/worksheets/README.md):
  printable school and university activities for atom cards, formula graphs, trait
  design, and CLI extension.
- [Visual language](https://github.com/shastaratech/bio-memory-rust-lab/blob/main/visuals/README.md):
  Mermaid and PlantUML diagrams for structures, algorithms, contracts, and flows.
- [Molecule Explorer CLI](https://github.com/shastaratech/bio-memory-rust-lab/blob/main/projects/molecule-explorer-cli.md):
  first capstone project for running molecule graph commands.
- [Molecule Explorer CLI handout](https://github.com/shastaratech/bio-memory-rust-lab/blob/main/handouts/molecule-explorer-cli.md):
  student worksheet for command prediction, code reading, and capstone extensions.
- [DNA memory handout](https://github.com/shastaratech/bio-memory-rust-lab/blob/main/handouts/dna-memory-data-structures.md):
  comparison of DNA sequence information with stack, heap, references, slices, and
  Rust data structures.
- [3D molecule viewer](https://github.com/shastaratech/bio-memory-rust-lab/blob/main/visuals/html/molecule-3d-viewer.html):
  interactive teaching view for molecules, graph edges, and approximate 3D shape.
- [Molecule design algorithms](https://github.com/shastaratech/bio-memory-rust-lab/blob/main/visuals/mermaid/molecule-design-algorithms.md):
  scaffold, candidate generation, filtering, scoring, ranking, and search frontier
  diagrams.
