# Worksheets

These worksheets turn the Bio Memory Rust Lab lessons into printable classroom
activities and focused university labs.

Use them alongside the book, lessons, and runnable Rust exercise.

## School Worksheets

- [Atom Cards](school-atom-cards.md): build a molecule with element cards, atom
  properties, and bond strings.
- [Formula To Graph](school-formula-to-graph.md): turn formulas into atom counts,
  then into molecule graphs.
- [DNA, Memory, and Data Structures](dna-memory-data-structures.md): compare base
  sequences, codons, gene slices, stack values, heap-backed vectors, and borrowed
  references.
- [Visualizing Molecules and Data Algorithms](visualizing-data-algorithms.md):
  choose arrays, lists, sorted lists, hash maps, graphs, and 3D coordinates for
  molecule questions.
- [Molecule Design As Data Search](molecule-design-data-search.md): move from a
  scaffold to candidate generation, validation, scoring, ranking, and the next
  design round.
- [Molecular Library Indexes](molecular-library-indexes.md): organize molecule
  records as vectors, hash maps, formula buckets, sorted lists, merged lists, and
  binary-searchable ranges.
- [Toy Fingerprints And Similarity Search](toy-fingerprints-similarity.md): turn
  molecule observations into feature bits, compare shared features, and rank
  nearest matches.
- [Screening Results And Feedback Loops](screening-feedback-loops.md): turn scores
  into test/defer decisions, compare predictions with experimental labels, and
  compute confusion-matrix metrics.
- [Serialization And Reproducible Lab Records](serialization-lab-records.md):
  serialize molecule summaries, parse records back, diagnose bad rows, and explain
  provenance.

## University Worksheets

- [Traits API Design](university-traits-api-design.md): decide which molecule
  behaviors belong in traits, methods, and free functions.
- [CLI Extension](university-cli-extension.md): extend the Molecule Explorer CLI
  with command parsing, a new command, and tests.

## Instructor Rule

Keep the worksheet output concrete.

Every worksheet should end with something students can show:

- a filled table
- a molecule graph
- a Rust function signature
- a command output
- a short explanation of what the model leaves out
