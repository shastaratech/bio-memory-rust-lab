# Projects

Projects turn the lessons into things students can run, modify, and explain.

## Suggested Path

1. [Molecule Explorer CLI](molecule-explorer-cli.md)
2. Formula calculator
3. Graph path finder
4. Molecule validator
5. Toy analog generator

The first project is already backed by runnable code in:

```text
exercises/rust-molecule-model
```

## Project Rules

- Use only toy molecules encoded in source code.
- Do not add raw molecular datasets.
- Keep functions small and readable.
- Every feature should connect back to a lesson.
- Every new algorithm should have at least one test.

## Presentation Format

Students should be able to explain:

1. What data structures they used.
2. What functions answer which questions.
3. What traits describe shared behavior.
4. What their model leaves out.
